# 🧠 Smart Notes App (React)

A modern and interactive **Smart Notes Application** built using React.
It allows users to create, manage, search, and organize notes efficiently.

---

## 🚀 Live Demo

👉 https://dhanusha-bh.github.io/smart-notes/

---

## ✨ Features

* ➕ Add new notes
* 👀 View all notes dynamically
* ✏️ Edit existing notes
* ❌ Delete notes
* ⭐ Mark notes as important
* 🔍 Search notes in real-time
* 🏷️ Categorize notes (Study, Work, Personal)
* 📱 Responsive design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

## ⚙️ Installation & Setup

Clone the repository and run locally:

```bash
git clone https://github.com/dhanusha-bh/smart-notes.git
cd smart-notes
npm install
npm start
```

---

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

```bash
npm run deploy
```
